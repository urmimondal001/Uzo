# Reels App Backend

This is the backend API for a Reels-type app built with Node.js, Supabase, and R2 Blaze.

## API Endpoints

- GET /api/reels → Fetch all reels
- POST /api/reels/upload → Upload a new reel (title, Base64 video, filename)

## Setup

1. Create `.env` with your Supabase & R2 credentials
2. Run `npm install`
3. Run `npm run dev` for development
